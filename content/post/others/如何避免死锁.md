# 产生的条件


+ 互斥：资源不能被共享，只能由一个进程使用。
+ 请求与保持：进程已获得了一些资源，但因请求其它资源被阻塞时，对已获得的资源保持不放。
+ 不可抢占：有些系统资源是不可抢占的，当某个进程已获得这种资源后，系统不能强行收回，只能由进程使用完时自己释放。
+ 循环等待：若干个进程形成环形链，每个都占用对方申请的下一个资源。
