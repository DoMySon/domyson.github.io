# 序

`Anywhere` 是一个基于 Unity 引擎的游戏框架，它设计理论源自于 `ECS` 当然还没有 `Natvie Code` 也没有 `Brust` 编译器。脱离 Unity 本身的编码方式，致力于采用全代码开发的方式



# 设计

像刚接触Unity那会，每次写完`Monobehaviour`脚本总是需要挂在到一个物体上，然后提交的时候一旦忘记提交`.meta`文件，那么就需要重新连连看，所以为了解决这个问题 采用动态绑定的方式去关联物体，全程没有任何显式`Monobehaviour`编程
