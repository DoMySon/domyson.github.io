# skynet定时器





# 简介

`timer` 模块是 `skynet` 内置的一个定时模块，使用了小根堆实现。



# 使用

`skynet.timeout(d,fn)` 注册一个到期时间，而之后将会触发这个函数，特别注意，此函数的执行协程同当前调用的协程，简单来说就是并不会造成资源竞争，但是需要注意一下 `aba` 问题


# QA

1. 为什么不提供循环计时器？

    你可以使用闭包来实现循环

2. 为什么不提供取消？

    你可以使用标记位来控制逻辑

3. 为什么使用小根堆

    为了确保精度，以及内存，简单而言其实是一个`topk`算法的变种，不需要在一次循环中做完所有事(10ms)
